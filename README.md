# Holiday Circuit Playground Express Adventure 🎄

An electronics advent calendar featuring 25 days of fun Circuit Playground Express projects for kids!

## 🎯 About This Project
This repository contains a collection of 25 holiday-themed electronics projects designed for children to learn about circuits, coding, and creativity. Each project is adapted for two age groups:
- Age 9: Basic circuit building and simple programming concepts
- Age 13: More advanced programming and electronics concepts

## 🚀 Getting Started

### Required Materials
Essential materials needed for all projects:
- Circuit Playground Express (main board)
- USB Cable (for programming)
- Mini Breadboard (used from Day 6 onwards)
- Jumper Wires and Alligator Clips
- LEDs (various colors)
- Copper Tape
- Coin Cell Batteries
- STEMMA QT / Qwiic cables (for sensors)
- Cardstock/Paper
- Basic Craft Supplies

Specific materials for each day's project are listed in the individual project instructions.

### Software Requirements
- CircuitPython 8.x installed on your Circuit Playground Express
- Code Editor (recommended: Mu Editor or VS Code with CircuitPython extension)
- Latest CircuitPython Libraries Bundle
- USB connection to your computer

## 🎓 Skill Progression

The projects follow a structured learning path:

1. **Basic Electronics (Days 1-3)**
   - Simple circuits and LED control
   - Basic switches and power
   - Introduction to circuit design

2. **Programming Introduction (Days 4-6)**
   - Circuit Playground Express basics
   - Basic coding concepts
   - NeoPixel programming

3. **Sound and Input (Days 7-9)**
   - Basic sound output
   - User input mechanisms
   - Interactive controls

4. **Sensors and Feedback (Days 10-14)**
   - Light and magnetic sensing
   - Environmental monitoring
   - Visual feedback systems

5. **Advanced Controls (Days 15-20)**
   - Remote control systems
   - Complex sensors
   - Multiple component integration

6. **Final Projects (Days 21-25)**
   - System integration
   - Advanced animations
   - Complex interactions
